**Xsharp is a programming language in development. It is mainly used for embedded systems.**

### Install
- Requeriments:
    - Linux (Archlinux, Ubuntu)
    - GCC
    - Bison(Optional)
    - 32 - 64 bits
    - 500MB RAM
    - Storage ... > 500MB
    - 1.8 Ghz
- Clone repository
- Execute build.sh
- ```bash
    $ sudo ./xsharp-compiler [file.xs]
    ```
